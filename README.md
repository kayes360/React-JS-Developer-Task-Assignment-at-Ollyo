
## Run Locally

Clone the project

```bash
  git clone https://github.com/kayes360/React-JS-Developer-Task-Assignment-at-Ollyo.git
  
```

Go to the project directory

```bash
  cd React-JS-Developer-Task-Assignment-at-Ollyo
```


Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
